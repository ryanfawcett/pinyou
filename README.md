### 项目整体介绍

- 项目名称：品优购
- 项目描述：品优购是一个电商网站, 该项目包含 PC 端首页、列表页以及注册页面的制作

### 学习目的

1. 电商类网站比较综合, 里面需要大量的**布局技术**, 包括布局方式、常见效果以及周边技术
2. 复习、总结 HTML、CSS 基础知识
3. 能对实际开发中制作 PC 端页面有一个整体的感知
4. 为后续学习移动端项目做铺垫

### 开发工具及技术栈

1. 开发工具

- Visual Studio Code
- Photoshop
- Chrome

2. 技术栈

- 利用 HTML5 + CSS3 手动布局, 可以大量使用 H5 新增标签和样式
- 采取结构与样式分离, 模块化开发
